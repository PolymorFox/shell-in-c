# A basic shell in c

This is a basic shell written entirely in c and cmake

## How to Build

```bash
# 1. Clone the repository
git clone [https://github.com/PolymorFox/shell-in-c.git](https://github.com/PolymorFox/shell-in-c.git)
cd shell-in-c

# 2. Make your build folder
mkdir build
cd build

# 3. Configure the project
cmake ..

# 4. Build and compile the executable
cmake --build .
```

# How to Run
Note: the executable will be located in the `build` directory:

```
./shell
```
